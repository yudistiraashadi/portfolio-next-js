
New e-Karyasiswa is the next-generation scholarship management system for BPSDM PUPR (Badan Pengembangan Sumber Daya Manusia, Ministry of Public Works and Housing). It rebuilds the existing Karyasiswa monitoring dashboard from the ground up with a modern full-stack architecture, delivering improved performance, type safety, and an enhanced user experience for managing scholarship recipients.

## Key Features

### 1. Scholarship Recipient Management

Comprehensive profiles for each scholarship recipient track personal information, academic institution, program of study, and funding details. Administrators can manage the full lifecycle from initial selection through graduation or program completion.

### 2. Study Progress Monitoring

Structured tracking of academic milestones — semester completion, GPA trends, thesis progress, and expected graduation dates. Supervisors receive clear visibility into each student's trajectory and can identify those falling behind schedule.

### 3. Periodic Reporting Workflows

Scholarship recipients submit structured progress reports through the platform on defined schedules. The system enforces reporting deadlines and provides administrators with a consistent, auditable record of student updates.

### 4. Document Management

Integration with S3-compatible object storage (MinIO) enables secure upload and retrieval of scholarship-related documents — acceptance letters, transcripts, progress reports, and completion certificates — all linked to individual recipient profiles.

### 5. Role-Based Access Control

SSO-capable authentication with role-based permissions ensures that students, supervisors, and administrators each see only what they need. Administrative functions are restricted to authorized personnel while students interact through a streamlined submission interface.

### 6. Type-Safe Full-Stack Architecture

Built on Next.js with oRPC for end-to-end type safety, Drizzle ORM for database operations, and Redis caching for responsive dashboards — ensuring reliability and maintainability as the system scales across BPSDM's scholarship programs.

## Purpose and Benefits

New e-Karyasiswa modernizes BPSDM PUPR's scholarship management infrastructure by replacing the legacy system with a performant, type-safe platform. The rebuild addresses technical debt while preserving the workflows that administrators rely on, delivering faster page loads, improved data integrity, and a foundation that can evolve with the ministry's needs.
