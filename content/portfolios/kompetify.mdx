
Kompetify is an adaptive managerial competency testing platform originally built for Indonesia's Ministry of Public Works and Housing (PUPR). In October 2025, BKD Jawa Timur (Badan Kepegawaian Daerah Provinsi Jawa Timur) signed a cooperation agreement to adopt Kompetify for assessing ASN competencies across provincial agencies. The system administers exams that dynamically adjust difficulty based on test-taker performance, providing precise competency measurements across nine managerial dimensions mandated for government officials.

## Key Features

### 1. Adaptive Exam Engine

Each exam consists of 27 questions dynamically selected based on competency sequence, indicator sequence, and adaptive difficulty levels ranging from 1 to 5. The system adjusts subsequent question difficulty based on previous answer weights, ensuring each test-taker receives appropriately challenging questions for accurate assessment.

### 2. Nine Competency Dimensions

Exams assess government-mandated managerial competencies: integritas, komunikasi, perekat bangsa, orientasi pada hasil, pelayanan publik, kerjasama, mengelola perubahan, pengambilan keputusan, and pengembangan diri dan orang lain. Each dimension is scored independently, producing a detailed competency profile.

### 3. Class and Batch Management

Organizers create exam classes linked to adaptive profiles, managing participant enrollment, scheduling, and access control. The batch system supports large-scale assessment events across multiple organizational units simultaneously.

### 4. Question Bank Management

A rich text editor with image support allows administrators to create and manage questions tagged with competency, indicator, and difficulty metadata. Multiple-choice options carry weighted scores that feed into the adaptive algorithm. Bulk import via CSV and Excel streamlines question bank population.

### 5. Reporting and Analytics

Visual reports built with interactive charts present individual competency profiles and aggregate assessment results. PDF report generation enables offline distribution, while dashboard analytics help administrators monitor exam completion rates and score distributions across cohorts.

## Purpose and Benefits

Kompetify replaces static, one-size-fits-all competency exams with adaptive assessments that accurately measure each official's managerial capabilities. By tailoring difficulty to individual performance, the platform produces more reliable competency data — supporting better decisions about training needs, role assignments, and professional development. Its adoption by BKD Jawa Timur alongside Kementerian PUPR demonstrates the platform's scalability beyond a single organization into a reusable assessment tool for government agencies nationwide.
