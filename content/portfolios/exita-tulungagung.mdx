Exita — short for Explorasi Tulungagung — is a mobile app that helps residents and visitors navigate public transportation in Tulungagung City, Indonesia. The app provides real-time route information, nearby stops, and map-based navigation for the city's angkot (minibus) network.

## Key Features

1. **Route Map** — Interactive GIS-powered map showing all public transportation routes with stop locations and directions.
2. **Nearby Stops** — Detects the user's location and surfaces the closest transportation stops within walking distance.
3. **Route Search** — Find the best route between two points using the local angkot network.
4. **Transportation Info** — Details on operating hours, fares, and route coverage for each line.

## Technology

Built with React Native and MobX for state management, with GIS integration for map rendering and geospatial features. The app targets both Android and iOS and was built in close coordination with the Tulungagung City transportation office to ensure route data accuracy.
